Promiseとは

PromiseとはES6から導入された概念で、 JavaScriptの非同期処理をコールバック地獄にならないようにかけます。 

        
          asyncFunction(success, error);
          function success(result){
            doSomething(result);
          }
と書いていたものを 
        
          asyncFunctionWithPromise().then(function(result){
            doSomething();
          }).catch(error);
のように記述できる。
